
# Setup

    $ git submodule add https://github.com/zwbetz-gh/cupper-hugo-theme.git themes/cupper-hugo-theme

# Update theme

    $ git submodule update --remote --merge

# Serve site locally

    $ cd hugo
    $ ./serve_local.sh

# Build site

    $ cd hugo
    $ ./publish.sh

The new doc files are in `jctest/docs`, ready to be checked in.
